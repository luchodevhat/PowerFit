PowerFit Store Web Application | THIS PROJECT IS STILL ON DEVELOPMENT

A full-stack web application for a sportswear store that allows users to browse products, create accounts, log in as customers or administrators, and place orders through WhatsApp integration.

📌 Project Overview

This project is an e-commerce web application built for a sports clothing store. It provides:

A modern and responsive user interface

User authentication system (Admin & Customer roles)

Product browsing and store section

WhatsApp integration for purchase orders

Backend API with database integration

The system is designed to simulate a real-world online store with role-based access and purchasing workflow.

💬 WhatsApp Integration

Customers can complete their purchases via WhatsApp.

When a user selects products and proceeds to checkout, the system generates a pre-filled WhatsApp message containing:

Product details

Total amount

Customer information

This allows direct communication with the store for order confirmation.
